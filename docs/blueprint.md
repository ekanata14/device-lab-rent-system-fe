# **App Name**: LabPrint Monitor Web

## Core Features:

- Real-time Printer Status Display: Visualize all 3D printers with their current status: available, in use (with a countdown timer), or broken. This will be the main dashboard view for lab users.
- Printer Usage Reservation: Allow users to book an available printer by inputting their name, phone number, ID, and desired usage duration.
- Broken Printer Reporting: Provide a quick and easy way for users to flag a specific printer as broken, taking it out of commission for an uncertain amount of time.
- Automated Countdown Timer: Display a dynamic countdown timer for printers currently in use, indicating how long until they become available.
- User Session Management: A backend system to store user reservation details and link them to specific printer sessions.

## Style Guidelines:

- Primary accent color (for interactive elements and highlights): A striking, cool medical blue (#1D90C9), reflecting a precise and technological vibe.
- Background color: A very dark, almost black grey with a subtle blue tint (#171A1C), providing a sleek, modern, and high-contrast base.
- Secondary accent color: A bright, clear aqua (#7DE8D6) that provides a fresh, complementary pop, used for secondary information or interactive elements.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, techy, and direct feel appropriate for a lab setting.
- Body font: 'Inter' (sans-serif) for clean, objective, and highly legible text that is easy to read across various display sizes.
- Use clean, minimalistic line-art icons to clearly denote printer status, reservation actions, and breakdown reporting.
- Employ a modular, card-based layout for individual printer displays, making status and actions easily discernible at a glance.
- Subtle and functional UI animations, such as gentle transitions for status updates or form submissions, to provide responsive feedback without distraction.
